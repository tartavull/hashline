# hashline (Codex skill)

A Codex skill + CLI for hash-verified, line-addressed edits.

- Skill: `hashline/` (contains `SKILL.md`)
- CLI: `hashline` (Rust; build instructions in the skill)

## Install

Using skills.sh tooling:

```bash
npx skills add https://github.com/tartavull/hashline --skill hashline
```

Or manually copy `hashline/` into `~/.codex/skills/`.
